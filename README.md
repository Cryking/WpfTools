# WPF 多功能工具箱

一个集成了 JSON 格式化和图片/Base64转换功能的 WPF 桌面应用程序。

## 功能特点

### JSON 工具
- JSON 字符串格式化（美化）
- JSON 字符串压缩
- JSON 语法验证
- 支持文件拖拽
- 支持搜索功能（Ctrl+F）

### 图片/Base64 转换工具
- 图片转 Base64
- Base64 转图片
- 支持常见图片格式：jpg, png, gif, bmp, ico, webp
- 支持从剪贴板粘贴图片
- 支持文件拖拽

### 系统托盘功能
- 最小化到系统托盘
- 右键菜单操作
- 程序后台运行

### 定时提醒功能
- 可自定义提醒间隔时间（分钟）
- 可自定义提醒内容
- 通过系统托盘通知和弹窗提醒
- 支持开始/停止提醒

## 使用说明

### JSON 工具
1. 在左侧输入框中输入或粘贴 JSON 字符串，或直接拖拽 JSON 文件
2. 点击"格式化"、"压缩"或"验证"按钮进行处理
3. 处理结果显示在右侧文本框中
4. 使用 Ctrl+F 可以在右侧文本框中查找内容

### 图片/Base64 转换
1. 通过"选择图片"按钮选择图片文件，或使用"粘贴图片"按钮从剪贴板获取图片
2. 点击"图片转Base64"将图片转换为Base64字符串
3. 点击"Base64转图片"将Base64字符串转换回图片
4. 可以使用复制按钮将结果复制到剪贴板

### 定时提醒
1. 点击"定时提醒"按钮打开提醒设置对话框
2. 设置提醒间隔时间（分钟）和提醒内容
3. 点击"开始提醒"按钮启动定时提醒
4. 点击"停止提醒"按钮停止定时提醒
5. 当窗口最小化到托盘时，也可以通过右键菜单访问提醒设置

## 重要提示：操作系统通知设置

为了使定时提醒功能正常工作，**请确保在操作系统中开启了通知设置**：

### Windows 10/11 设置步骤：
1. 打开 Windows 设置 (快捷键: Win + I)
2. 点击"系统" > "通知和操作"
3. 确保"获取来自应用和其他发送者的通知"已开启
4. 在应用通知列表中找到本应用，确保其通知权限已开启

### 其他注意事项：
- 如果系统通知未开启，定时提醒仍会通过弹窗方式显示，但不会显示托盘气泡通知
- 程序最小化到系统托盘后，仍会继续运行定时提醒
- 关闭窗口不会退出程序，而是最小化到托盘

## 系统要求

- Windows 10/11
- .NET 8.0 或更高版本

## 构建和运行

1. 克隆或下载项目代码
2. 使用 Visual Studio 2022 或更高版本打开解决方案文件
3. 按下 F5 或点击"启动"按钮运行程序

或者使用命令行：

```bash
dotnet restore
dotnet build
dotnet run
```
